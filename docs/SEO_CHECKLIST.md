# SEO Checklist for Editors

- [ ] **Unique Title:** `[Property Name] | RENT 2026`
- [ ] **Meta Description:** 150-160 znakov s kľúčovými slovami.
- [ ] **Alt Text:** Popisný text pre všetky obrázky.
- [ ] **Heading Structure:** Iba jeden `<h1>` na stránku.
- [ ] **Canonical URL:** Skontrolovať duplicitu.
