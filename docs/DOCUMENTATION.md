# RENT Dashboard 2026

> The next-generation CRM and Property Management System designed for the Thai Real Estate Market.

## Getting Started
1. **Clone:** `git clone https://github.com/your-repo/rent-dashboard-2026.git`
2. **Install:** `npm install`
3. **Run:** `npm run dev`

## Architecture
- **Frontend:** React + Vite
- **Backend:** Lovable Cloud (Supabase)
- **State:** Zustand + TanStack Query
- **Styling:** Tailwind CSS

## Features
- Property Management
- Lead Pipeline (Kanban)
- Payment Tracking
- Real-time Notifications
- Multi-language Support (EN/SK/TH)
