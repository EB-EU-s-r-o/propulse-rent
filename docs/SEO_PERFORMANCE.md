# SEO & Performance Strategy

| Kategória | Úloha | Detail / Nástroj |
| :--- | :--- | :--- |
| **SEO: Meta Data** | Dynamické `<title>` & Meta | Implementovať `react-helmet-async` pre každú stránku nehnuteľnosti. |
| **SEO: Schema** | Structured Data (JSON-LD) | Pridať `schema.org/ApartmentComplex`. |
| **SEO: Sitemap** | Generovanie Sitemap.xml | Použiť `vite-plugin-sitemap`. |
| **Perf: Images** | Image Optimization | Použiť optimalizované formáty (WebP) s `priority` pre LCP obrázky. |
| **Perf: Lazy Load** | Code Splitting | Použiť `React.lazy` pre mapy a grafy. |
| **Perf: Fonts** | Font Optimalizácia | Preload pre 'Plus Jakarta Sans' a 'JetBrains Mono'. |
